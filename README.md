# 호이짱 트래블 ✈️

여행사 웹사이트 프로젝트

## 📋 프로젝트 소개

호이짱 트래블은 여행 서비스를 제공하는 웹사이트입니다. 사용자 인증 기능과 함께 여행 상품 안내, Q&A, 문의하기 등의 기능을 제공합니다.

## 실행하기 
깃배쉬에서 npm run dev 
입력 후 http://localhost:5174/ 클릭

## 🚀 주요 기능

### 1. 메인 페이지
- 반응형 네비게이션 바
- 히어로 섹션 (메인 배너)
- 회사 소개 섹션
- 서비스 안내 (6가지 서비스)
- 문의하기 섹션
- 부드러운 스크롤 애니메이션

### 2. 회원 인증 시스템
- **회원가입**: 이름, 이메일, 비밀번호로 가입
- **로그인**: 이메일/비밀번호 인증
- **로그아웃**: 로그인 상태 해제
- **세션 관리**: localStorage 기반 인증 유지

### 3. Q&A 페이지
- 8개 카테고리별 FAQ
  - 예약 (예약 방법, 취소/환불)
  - 상품 (패키지, 맞춤형 여행)
  - 결제
  - 비자
  - 현지정보
  - 기타
- 카테고리 배지 표시
- 호버 애니메이션 효과

## 💻 사용 방법

### 1. 프로젝트 실행
```bash
# 파일 탐색기에서 travel-agency-website.html을 브라우저로 열기
# 또는
cd travel
start travel-agency-website.html  # Windows
open travel-agency-website.html   # macOS
```

### 2. 회원가입
1. 메인 페이지 우측 상단 "로그인" 버튼 클릭
2. "회원가입" 탭 선택
3. 정보 입력 (이름, 이메일, 비밀번호)
4. "회원가입" 버튼 클릭

### 3. 로그인
1. 로그인 페이지에서 이메일/비밀번호 입력
2. "로그인" 버튼 클릭
3. 메인 페이지로 자동 이동
4. 우측 상단에 "👤 이름님" 표시

### 4. 로그아웃
1. 우측 상단 "로그아웃" 버튼 클릭
2. 확인 후 로그아웃 완료

## 📊 데이터 저장 구조

### localStorage 키
- **`users`**: 전체 회원 정보 배열
- **`currentUser`**: 현재 로그인한 사용자 정보

### 데이터 예시
```javascript
// users
[
  {
    "name": "홍길동",
    "email": "user@example.com",
    "password": "12345678",
    "registeredAt": "2024-11-04T04:15:30.000Z"
  }
]

// currentUser
{
  "name": "홍길동",
  "email": "user@example.com",
  "loginAt": "2024-11-04T05:20:15.000Z"
}
```

## ⚠️ 주의사항

### 보안 관련
> **이 프로젝트는 학습 및 테스트 목적으로만 사용하세요!**

**현재 구현의 한계:**
- ❌ 비밀번호가 평문(plain text)으로 저장됨
- ❌ XSS, CSRF 등 보안 취약점 존재
- ❌ 서버 검증 없음 (클라이언트 측만 검증)
- ❌ HTTPS 미적용
- ❌ 브라우저 localStorage는 쉽게 접근 가능

**실제 서비스 개발 시 필요한 사항:**
- ✅ 백엔드 서버 (Node.js, Python 등)
- ✅ 데이터베이스 (MySQL, PostgreSQL, MongoDB 등)
- ✅ 비밀번호 해싱 (bcrypt, argon2 등)
- ✅ HTTPS 적용
- ✅ JWT 또는 세션 기반 인증
- ✅ 입력값 서버 측 검증
- ✅ CORS 설정
- ✅ Rate Limiting
